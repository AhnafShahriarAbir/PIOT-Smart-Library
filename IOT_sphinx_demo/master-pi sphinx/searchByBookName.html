searchByBookName Doc¶

class searchByBookName.VoiceSearch[source]¶

This class is used to look for a book by using voice. The class is called in library_menu, when an user wants to search for a book by using his/her voice. author: @shahriar_abir

main()[source]¶

This method calls the getBookNameToSearch first and assigns the returned value to bookName. If bookName is none, then prints “Failed to get the book”

Once the bookName is found, the searchBook(bookName) is called which takes the bookName as the parameter we got from the first method.

If book is found, prints the result. author: @shahriar_abir

searchBook(bookName)[source]¶

This method takes bookName as a parameter and searches for that book in database after the connection is made by using MySQLdb.connect. author: @shahriar_abir
