if you want to know spinx

1. pip3 install sphinx
2. mkdir demo
3. In demo directory,create two folder named:doc and src,src is to store sample demo code (will be our    code)
3. cd demo
4. sphinx-quicksatrt
5. fill up information
6. Go to the demo/doc folder ,change config.py ,mine version is uncomment the line 13-15,and add extension as below:

extensions = [
    'sphinx.ext.autodoc',
    'sphinx.ext.doctest',
    'sphinx.ext.intersphinx',
    'sphinx.ext.todo',
    'sphinx.ext.coverage',
    'sphinx.ext.mathjax'

]
7. sphinx-apidoc -o sphinx_demo/source ../src/
8. cd .
9. sphinx_demo>sphinx-build -b html source build
10. make html
